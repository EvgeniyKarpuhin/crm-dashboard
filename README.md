CRM Dashboard (React + TypeScript)

A modern CRM dashboard application for managing clients and tasks.

🚀 Tech Stack
React (Vite)
TypeScript
Zustand (state management)
React Router
Tailwind CSS
Axios

✨ Features
Dashboard with statistics
Task management (kanban board: create, update, drag & drop)
Client management (CRUD operations)
Task filtering by client
Global state management with Zustand
API integration (REST, Axios)
Responsive UI (Tailwind CSS)
Modal windows and form handling

📦 Installation
npm install
npm run dev
