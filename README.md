CRM Dashboard (React + TypeScript)

A modern CRM dashboard application for managing clients and tasks.

🚀 Tech Stack
React (Vite)
TypeScript
Zustand (state management)
React Router
Tailwind CSS
Axios

✨ Features
Dashboard with statistics
Task management (create, update, filter)
Client management
Responsive UI

📦 Installation
npm install
npm run dev
